# BestGUIProjectEver

![Alt text](https://github.com/emma4thompson/BestGUIProjectEver/blob/master/logo2.png) </br>

1. Install Qt 5.6 </br> 
2. Open project </br> 
3. Clean all, run qmake, build all, run.  </br>
